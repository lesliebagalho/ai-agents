## Requisitos Iniciais

- O sistema deve ser web.
- O sistema deve suportar multiusuário.
- O sistema deve suportar multiempresa.
- O sistema deve permitir evolução para SaaS ou sistema interno.
- O sistema deve seguir a stack padrão do framework, salvo decisão futura.
- O sistema deve manter rastreabilidade básica das movimentações.
- O sistema deve preservar isolamento de dados entre empresas.

## Pendências para Discovery

- Definir metas de performance.
- Definir política de auditoria.
- Definir requisitos de segurança e backup.
