## Requisitos Iniciais

- O sistema deve permitir cadastrar produtos.
- O sistema deve permitir cadastrar categorias.
- O sistema deve permitir controlar entradas de estoque.
- O sistema deve permitir controlar saídas de estoque.
- O sistema deve permitir ajuste manual de estoque.
- O sistema deve permitir consultar saldo atual.
- O sistema deve permitir visualizar histórico de movimentações.
- O sistema deve permitir busca e filtros básicos de produtos.
- O sistema deve suportar múltiplos usuários.
- O sistema deve suportar múltiplas empresas com separação de dados.
- O sistema deve permitir perfis e permissões básicas por usuário.
